# Privacy Policy for Gmail Unread Filter

This extension does not collect, store, or transmit any personal information, browsing history, or user data. All functionality runs locally in your browser.

## Permissions Justification

This extension requires the following permission:

*   **Host Permission for `https://mail.google.com/*`**: This permission is necessary to allow the extension to add the "Unread" button to the Gmail user interface. The extension does not interact with any other part of the page or your data.
